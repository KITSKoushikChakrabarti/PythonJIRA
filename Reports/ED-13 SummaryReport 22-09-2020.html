Description+*Objective:*+  Flexible Digital Environments programme intends to provide the ability to spin up any configuration of the digital stack and its components, with assured data integrity, a set of automated tests, mocked third-party endpoints and canary monitoring; in order for us to be able to test any integration combination of any application version across Digital.   +Benefits:+ * Reduce contention in the environments, thereby leading to faster release to market * Improve traceability of changes and maintenance of control on environments * Leverage DevOps capabilities for on-demand environment spin-up * Improve predictability of Test Results by maintaining configuration state during test cycle, reduce unnecessary errors * Reduce environment maintenance costs by implementing 'spin-up-as-required' approach instead of 'always-on'
